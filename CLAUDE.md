## コミットメッセージのルール

コミットメッセージは必ず以下の形式で書くこと：

```
【課題】何の問題を解決したか
【解決】どう解決したか
【指示】Claudeへの指示内容（プロンプトの要約）
```

### 具体例

```
【課題】ログイン画面のバリデーションが未実装【解決】zod + react-hook-formで入力チェックを実装【指示】zodスキーマを定義してreact-hook-formのresolveに渡してバリデーションを実装せよ
```

### ルール

- `【】`の中の改行は不可。各項目は1行で書くこと
- 3つのタグがすべて存在する場合のみNotionに自動蓄積される
- この形式に従うことで、GitHub ActionsがNotionに自動蓄積し、毎日20:00 JSTにXへ自動投稿される
